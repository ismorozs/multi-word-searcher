How to use

Press ENTER when focused on input to create a new input for entering an additional substring
Set a number in a smaller input that will correspond to a maximum allowed distance in characters for searcing between two substrings
Press BACKSPACE on empty input to remove it
If empty input loses focus it's removed as well
